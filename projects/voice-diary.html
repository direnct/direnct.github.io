Voice Diary App

A native Android application with real-time audio visualization.

Role: Lead Developer Platform: Android Year: 2026

Overview

Voice Diary is a personal journaling application that focuses on the power of the human voice. It allows users to capture their thoughts and emotions through high-quality audio recordings, which are then organized in an intuitive calendar-based interface.

Key Features

  • Real-time sound-reactive audio visualization.
  • Interactive calendar view for easy navigation of entries.
  • High-fidelity audio recording and playback.
  • Themed UI with glassmorphism elements.

Technologies

Kotlin Android SDK Jetpack Compose MediaRecorder API
