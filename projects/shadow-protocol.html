Shadow Protocol

A fast-paced cyber-stealth action game built with Unity.

Role: Game Designer & Developer Engine: Unity 2023 Year: 2026

Overview

Shadow Protocol is a 3rd person stealth-action game set in a neon-drenched cyberpunk future. Players take on the role of an elite infiltrator using advanced hacking techniques and agile movement to navigate through high-security corporate installations.

Key Features

  • Custom Shader Graph for holographic and stealth effects.
  • Advanced AI behavior trees for guard patrols and detection.
  • Dynamic lighting system that impacts gameplay (stealth mechanics).
  • Fluid movement system including wall-running and dashing.

Technologies

Unity C# Shader Graph Universal Render Pipeline
